.. _install:

Install
------------------------------------------------------------------------------

``afwf_lastpass`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install afwf_lastpass

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade afwf_lastpass
